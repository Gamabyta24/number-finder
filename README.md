# Number-finder
[![Python](https://img.shields.io/badge/python-3.10-blue.svg)](https://www.python.org/downloads/release/python-3100/)
[![flake8](https://img.shields.io/badge/flake8-passing-brightgreen)](https://flake8.pycqa.org)


Быстрый поиск кода ОКВЭД по окончанию номера телефона.

## Описание

Утилита находит код ОКВЭД, который максимально совпадает с окончанием номера телефона. Использует индексацию по длине кодов для мгновенного поиска среди 12 000+ записей.

### Как это работает

```
Телефон: +7 920 047-27-21
                   ↓
Нормализация: 79200472721
                   ↓
Поиск окончания: ...2721 → Код 27.21 ✓
```

## Требования

- Python 3.10+
- Файл `okved.json` с кодами ОКВЭД

## Установка

```bash
git clone https://github.com/Gamabyta24/number-finder.git
cd number-finder
```

Дополнительных зависимостей не требуется — только стандартная библиотека Python.

## Использование

### Командная строка

```bash
python okved_matcher.py
```

Программа запросит номер телефона:

```
Введите номер телефона: +79XXXXXXXXXX
```

И выведет результат:

```
Номер:            79XXXXX2721
Код ОКВЭД:        27.21
Название:         Производство электродвигателей
Длина совпадения: 4
```

## Формат файла ОКВЭД

Ожидается JSON с вложенной структурой:

```json
[
  {
    "code": "10.86",
    "name": "Производство детского питания",
    "items": [
      {
        "code": "10.86.1",
        "name": "Производство молочных продуктов для детей"
      }
    ]
  }
]
